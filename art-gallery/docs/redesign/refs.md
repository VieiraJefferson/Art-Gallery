# Referências de Design - Pallas Galaxy

## Inspirações Visuais

### Dribbble References
- Galerias de arte contemporânea
- Dark mode elegante com acentos vibrantes
- Tipografia expressiva e espaçamento generoso
- Efeitos de hover sutis e transições fluidas

### Estética Geral
- **Mood**: Surrealista, onírico, sofisticado
- **Paleta**: Escura com acentos de magenta/rosa vibrante
- **Tipografia**: Display fonts para títulos, sans-serif clean para corpo
- **Imagens**: Full-bleed, high-contrast, dramaticamente iluminadas

## Referências de UI Components (21st.dev style)

### Buttons
- Primary: Sólido com cor de acento
- Secondary: Outline com borda sutil
- Ghost: Apenas texto com underline animado no hover

### Cards
- Fundo surface com sombra suave
- Hover com elevação e scale sutil
- Imagem com aspect-ratio consistente

### Navigation
- Minimalista, transparente sobre hero
- Logo à esquerda, links à direita
- Mobile: Sidebar com animação slide-in

## Notas
- Manter consistência visual entre todas as páginas
- Priorizar performance de imagens (lazy loading, srcset)
- Acessibilidade: contrast ratio mínimo 4.5:1
